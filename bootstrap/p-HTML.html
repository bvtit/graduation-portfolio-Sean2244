Description

In this project, I was tasked to add features and make improvements to "Gerry's" website. This project lasted for a few months because we were continually adding features. As shown by the images to the left, we progessively added features such as the the id attributes in the Glossary section or the Contest tab with several sub-features. I used my recently acquired skills of creating navilgation lists, hypertext linking, utilzing IDs, imaging maping, hotspots, etc. The purpose of this project was not only to test my knowledge in HTML, but to simulate what it would be like to be contracted to work as a web designer.

Details

  • I used an online program called "Thimble" to create the website.
  • I was provided with a template CSS file; however, I was able to make changes to it as I saw fit.
